Lensa - UX Engineer

Instead of dividing the designer and front-end teams, we approached a new way of at Lensa. By integrating front-end developers (UX engineers, as we called ourselves) into the design team we improved the efficiency of designer-developer communication and made sure that the developed components were the same as the designers imagined them in our design system.

Team:
2 product designers and 2 ux-engineers.

Component system - web components

Problems

  • There's usually a communication gap between designers and developers
  • Developers have no resources for developing the component system
  • Different javascript frameworks have different component requirements
  • Developers are not aware of the existing components

Solution

  • Integrate developers into the design team
  • Create framework-agnostic components
  • UX engineers are responsible for the look and feel and technical compatibility of the components

Technical background

  • We used the Lit web component framework to create our web components
  • We used Storybook to test and communicate the components to the other teams (React and Svelte teams)
  • We also implemented Tailwind to speed up our development process
