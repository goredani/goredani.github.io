OptiMonk

OptiMonk is a free conversion optimization tool that turns your traffic into sales. It allows you to create, personalize and launch popups, sidebars, notification bars and fullscreens without coding. As a product designer, I was responsible for designing the new features and maintain, update the design system.

Team
2 product designers working with UX research and several front-end development teams.

Coupon manager with Shopify

Problem
Most of our clients were using Shopify for their webshops. They already had coupon codes in their account and we wanted to sync those coupon codes with Optimonk for a better user experience and a faster workflow.

Tasks

  • Enable coupon syncing with Shopify
  • Update the editor UI
  • Handle all of the edge cases

Constraints
We had to consider that Shopify provided limited API functionalities.

Conclusion
Due to the constraints we had to drop some functionalities, however we managed make the coupon code sync as pleasant as possible.

Visitor Property segmentation tool

Problem
Some of our bigger clients demanded more sophisticated segmentation options in order to target their campaigns more precisely.

Solution

  • Provide custom segmentation options
  • Make it possible to combine (AND/OR) these segmentation rules

Conclusion
After some iterations, we managed to deliver a simple, yet scalable rule editor, where smaller and bigger clients could edit their segmentation rules.
