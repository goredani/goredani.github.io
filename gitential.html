Gitential

The Gitential application has been created to make software development measurable and enable data-driven improvements across your projects and teams by focusing on 4 pillars of Code Quality, Team Collaboration, Cost-Effective Project Deliveries and On-Time Software Releases.

Team
Single designer with a front-end development team.

Dashboard

Problem
Create an interface for the MVP of the product so our first customers could use the service.

Tasks

  • Create the user flow
  • Create wireframes
  • Create the UI
  • Create a seamless onboarding so our users can integrate their repositories
  • Test the UI with our first clients

Constraints
Some of our features were incomplete and we had to rush to make it to market in time.

Conclusion
Most of our users were not aware of the metrics and how we are calculating them. (eg. performance) We had to find a more understandable way to provide this information to them.
